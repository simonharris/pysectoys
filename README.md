# pysectoys

A few toy programs implementing computer security-related algorithms in Python. Mainly for my own education.

### Algorithms include:
- Playfair Cipher
- Rivest Cipher 4 (RC4)
- Diffie-Hellman handshake
- Linear Congruential Generator
- Combined Linear Congruential Generator
